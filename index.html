Information

The Web-Based Wi-Fi Hotspot User Authentication and Access Control System was created specifically for Cebu Eastern College (CEC) to improve and secure the school’s internet connectivity management. The institution faced recurring issues such as unauthorized access, slow network performance due to uncontrolled usage, and the lack of a proper system to identify and manage users connecting to the school Wi-Fi. These problems often caused inconvenience for students, teachers, and staff who relied on stable internet access for learning and administrative purposes.

To address these challenges, the system was designed to automate user access, improve network control, and strengthen data security within the campus. It introduces a voucher-based login system, where administrators can issue unique access credentials to each student or staff member. This ensures that only authorized users can connect to the Wi-Fi network, while also allowing administrators to monitor user sessions, bandwidth usage, and connection duration in real time.

The creation of this system reflects CEC’s commitment to modernizing its technological infrastructure and promoting a more efficient, secure, and organized digital environment for both academic and administrative operations. By implementing this solution, the school can maximize its network resources, maintain fair access for all users, and support digital learning in a controlled and reliable manner.
